Soal Persamaan Garis Lurus

Sebuah garis melalui titik (0, 2) dan memiliki gradien m = 4.
Tentukan persamaan garis tersebut dalam bentuk y = mx + c.


Penjelasan:
Bentuk umum persamaan garis: y = mx + c
Diketahui: m = 4 dan garis melalui (0, 2)

Substitusi titik ke persamaan:
2 = 4(0) + c → c = 2

Jadi, persamaan garisnya adalah: y = 4x + 2.
